** Описание Решения КР:

1. В самом начале создается первоначальный массив (strs) с заданными строками.

2. Далее происходит вызов метода (newArray), который возвращает искомый массив строк длины <= 3

3. В начале данный метод считает количество строк с длинной <= 3

4. Затем инициализируется массив соответствующей длинны (равный count)

5. Следом в цикле foreach в новый массив заносятся элементы (строки) необходимой длины из старого массива

6. В конце метод возвращает новый массив строк newStrs

7. Далее в методе (main) осуществляется печать элементов искомого массива строк (newArray)
